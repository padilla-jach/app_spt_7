# Proyecto de Análisis de Datos de Vehículos

Este proyecto tiene como objetivo poner en práctica tareas habituales de ingeniería de software y análisis de datos. Se ha desarrollado una aplicación web con Streamlit que permite visualizar un histograma y un gráfico de dispersión interactivos.

## Funcionalidades de la aplicación web

- Encabezado con descripción del proyecto.
- Botón para generar un histograma.
- Botón para generar un gráfico de dispersión.
